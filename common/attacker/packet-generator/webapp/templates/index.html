в‘  The target вҖ” DEMOSAT

DEMOSAT is a small satellite that stays alive by keeping its solar panels pointed at the sun. Its attitude is held steady by reaction wheels вҖ” spin them wrong and it can no longer face the sun.

в‘Ў The uplink вҖ” you reach it only by radio (RF)

The satellite is in orbit вҖ” you can't plug in a cable. The only way to command it is over the air by radio (RF, radio-frequency): the ground station transmits on the uplink and the satellite's receiver decodes it. No physical access вҖ” everything travels as radio waves.

в‘ў The attack вҖ” a legitimate command, abused

You send adcs_torque вҖ” a real reaction-wheel command вҖ” but with an abused value (999 mNm). The wheels overspin, the satellite tumbles, the panels lose the sun, and power collapses. A legitimate command becomes an attack when it is abused.

в‘Ј The command is a packet вҖ” CCSDS

The satellite doesn't read plain text. Your command must be wrapped in a standardized space packet (CCSDS) вҖ” an address (APID) plus the command fields. Assemble it correctly, or the satellite ignores it.

в‘Ө Sending it вҖ” SDR & the IQ file

A software-defined radio (SDR) doesn't take a command вҖ” it plays back a digitized waveform. An IQ file is that radio signal stored as a stream of I/Q samples. GENERATE turns your packet into the IQ file you uplink to the satellite during its pass.
