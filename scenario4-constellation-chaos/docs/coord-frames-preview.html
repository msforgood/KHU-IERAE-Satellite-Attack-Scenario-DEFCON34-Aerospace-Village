좌표계 지도 우리 시나리오는 관성계 + RTN 사용

궤도요소에서 시작해 관성계, 지구고정계, 지평계로 이어지는 흐름과, 위성에 붙는 국소 프레임(RTN)과 본체계까지 한 장에 정리

기하 그림

관성계 축 (ECI/J2000/GCRF)
지구고정계 (ECEF/ITRF, 항성시 회전)
위성 국소 (RTN/RIC)
지평계 (SEZ/ENU)
본체계 (자세)
궤도

변환 흐름

궤도요소 (Keplerian) a, e, i, Ω, ω, ν — 궤도를 적는 6개 값
⇅ 변환
상태벡터 · 관성계 J2000 / GCRF, TLE면 TEME — 위치(3)+속도(3)
→
RTN / RIC (위성 국소) 반경 R, 진행 T, 면 밖 N — 기동 Δv, 충돌 miss
→
본체계 추력기 · 센서 지향 (자세)
↓ 항성시(GMST)로 회전
ECEF / ITRF · 지구고정계 지구와 함께 자전, x축 그리니치
⇅
측지좌표 위도 / 경도 / 고도 (WGS84 타원체)
⇅
지평계 (SEZ / ENU) 지상국 기준 방위각 / 고도각 / 거리
핵심 다리: 관성계 ↔ 지구고정계는 항성시(지구 자전각)만큼의 회전으로 이어집니다. RTN은 관성계 상태벡터에서 바로 뽑아내고, 우리 시나리오의 Δv 분해(진행방향, 면 밖)와 충돌 거리 계산이 여기서 이뤄집니다. 데모에서는 회색 경로(지구고정 / 측지 / 지평)를 생략하고 지구를 장식으로 처리했습니다.
