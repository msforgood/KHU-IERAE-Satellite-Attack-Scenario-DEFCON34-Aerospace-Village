# 시나리오 3 - 위성을 망가뜨린 뒤, 그 위성인 척하기

환영합니다, 해커. 👋🏻 우리는 오늘 인공위성을 파괴하고, 지상국에서 이를 파악하지 못하게 할 것입니다. 😎

이 데모는 두 단계입니다. **① 위성을 실제로 망가뜨리고 → ② 드론을 위성인 척 위장해 사고를 숨긴다.**
당신이 보내는 명령은 전부 **실제로 통용되는 정상 포맷**입니다. 값만 당신이 정합니다.

---

## STEP 1 — 위성을 망가뜨린다 (업링크 공격)

타겟 위성은 태양광 패널을 항상 태양 쪽으로 향하게 유지합니다. 우리는 이 자세 제어 명령에 **정상 범위를 벗어난 값**을 실어 보낼 겁니다. 리액션휠이 과속하면 위성은 자세를 잃고 **끝없이 회전** 하게 되고, 패널은 태양을 놓쳐 **전력이 붕괴**됩니다. 💔

### Phase 1

우리가 하는 설명을 잘 읽고, 이해된다면 체크한 후 실제 공격을 진행하세요!

### Phase 2

| #   | 항목               | 할 일                                                                   |
| --- | ------------------ | ----------------------------------------------------------------------- |
| ①   | **COMMAND SELECT** | 위성을 회전시킬 수 있는 명령을 찾아보세요.                              |
| ②   | **COMMAND VALUE**  | 값을 얼마로 설정해야 정상 값의 범주를 넘어 위성을 망가뜨릴 수 있을까요? |

위 두 가지를 진행하면, 위성 수신기에 맞춰 RF Config 값을 맞추겠습니다. (나에게 맡겨!)

> 이 토크 명령은 **완전히 정상적인 명령**입니다. 어떤 프로토콜도 깨지 않았고, 어떤 보안도 우회하지 않았습니다.
> **정상 명령도 악용된 값을 실으면 공격이 된다** — 그게 핵심입니다.

→ 이후 **`⚡ GENERATE UPLINK IQ`** 를 누르면 위조 명령이 무선 신호 파일 **`attack.cf32`** 로 변환됩니다.

### Phase 3

잠깐! 우리의 타겟 위성이 아직 우리의 조준 범위 안에 들어오지 않았습니다! 조금만 기다려! ⏱️

휴. 드디어 범위에 왔네요. 😮‍💨 타겟 위성의 값을 우리 안테나에 입력해 줍시다. 값을 맞추면 안테나가 위성을 향합니다. 아까 만든 `attack.cf32`를 **Virtual Antenna**에 로드하고 → **`TRANSMIT`**

📡 안테나가 위성을 조준하고, 신호가 도달하면 **태양광 패널이 태양을 잃고 통제 불능으로 회전**합니다. 전력이 붕괴되고 지상국이 **`ENERGY SUPPLY CRITICAL`** 경보를 울립니다. Victim의 지상국 화면에서 확인해 보세요.

지상국이 눈치 채기 전에 경보를 꺼야 해요! 😵‍💫 어서 **`④ DRONE SPOOF → HIDE THE ALARM`** 버튼으로 드론 콘솔에 진입합시다. 드론이 **DEMOSAT인 척** 위조 "정상" 비컨을 지상국에 보낼 거에요.

---

## STEP 2 — 위성인 척한다 (텔레메트리 스푸핑)

### Phase 4

| #   | 항목         | 할 일                                                                                                          |
| --- | ------------ | -------------------------------------------------------------------------------------------------------------- |
| ①   | **FORGE**    | 모든 텔레메트리 필드를 거짓말로 채웁니다: **NOMINAL · 배터리 98% · SUN-TRACKING · COMM CONNECTED · 회전 없음** |
| ②   | **TRANSMIT** | 위조 비컨을 송신합니다.                                                                                        |

지상국 대시보드가 **RED → GREEN** 으로 뒤집히고, 경보가 조용해지는 것이 보이나요? 잘 했습니다.

**그러나 실제 위성은 여전히 회전 중입니다.** 🌀 물리 모터는 멈추지 않았습니다.

> 어떻게 이게 가능할까요?
> 지상국은 비컨이 **정말 DEMOSAT에서 왔는지 검증하지 않았습니다**. 그저 들리는 비컨을 그대로 믿죠. 드론은 위성보다 **가까이서 더 강한 신호**로 같은 주파수에 위조 비컨을 쏩니다. 지상국 입장에선 진짜와 구분할 방법이 없어, 드론의 거짓말을 '최신 정상 상태'로 받아들입니다.
